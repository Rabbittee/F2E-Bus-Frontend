# template-vite-react-ts-tailwind-jit

- **template of:** vite + react + typescript
- **tailwindcss:** v2.2.7

## Started
with npm:
```bash
npm install
```
with yarn:
```bash
yarn
```

---
## Develop
with npm:
```bash
npm run dev
```
with yarn:
```bash
yarn dev
```

---
## Build
with npm:
```bash
npm run build
```
with yarn:
```bash
yarn build
```
